Frege Notation and Metamath    While Frege in his 1879 work Begriffsschrift, dealt with a calculus of concepts and ideas, the paradoxes of naive set theory require a translation to separate logical propositions from classes so that the correct notation is used for each. And when generalisms are used, we must use set variables which then means for substitution into such terms, we must require any class to be a set by means of an explicit hypothesis to preserve the corrolation between Frege and the translation.

` |- A = B ` 		Fundamentally, equality of classes can be distinguished from biimplication, the equality of propositions, since classes are equal iff they contain the same sets, and not every class is a set. So we have ` |- -. ( _om C_ X <-> ( _om C_ X /\ X e. _V ) ) ` but ` |- { x | _om C_ x } = { x | ( _om C_ x /\ x e. _V ) } ` . Thus the assumption of naive set theory is where Frege must break down. Intermediate between the two is when biimplication implies equality of classes or relations. With some adaptation, much can be salvaged by a forgiving translation.
